Setup Google Sheets Sync
Apps Script code to paste
›
1
Go to script.google.com → New project → delete all code → paste below
const SHEET_NAME='CineStreamDB'; function doGet(e){ const ss=SpreadsheetApp.getActiveSpreadsheet(); let sh=ss.getSheetByName(SHEET_NAME)||ss.insertSheet(SHEET_NAME); const d=sh.getRange(1,1).getValue(); return ContentService.createTextOutput(d||'{"movies":[],"cfg":{}}') .setMimeType(ContentService.MimeType.JSON); } function doPost(e){ const ss=SpreadsheetApp.getActiveSpreadsheet(); let sh=ss.getSheetByName(SHEET_NAME)||ss.insertSheet(SHEET_NAME); sh.getRange(1,1).setValue(e.postData.contents); return ContentService.createTextOutput('{"ok":true}') .setMimeType(ContentService.MimeType.JSON); }
2
Deploy → New deployment → Web app. Execute as: Me. Who has access: Anyone. Click Deploy → Authorize.
3
Copy the Web app URL → paste in Settings → Sync → Google Sheets → Save & Test.

How to use GitHub Gist Sync
Add movies on phone → see on laptop
›
1
Go to github.com/settings/tokens → Tokens (classic) → Generate new → Name: CineStream, Expiration: No expiration, check only: gist → Generate → Copy token.
2
Settings → Sync → GitHub Gist → paste token. Gist ID is pre-filled. Click Save & Test.
3
On same device: click ⬆ Push All to save library to cloud. On any other device: paste same token → click 🔄 Sync Now.
